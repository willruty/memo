# Memo - Sistema de Memórias

Sistema web para guardar memórias de eventos, viagens e festas.

## Tecnologias

- Backend: Python Flask
- Frontend: HTML5, CSS3
- Banco de Dados: PostgreSQL (Supabase)
- Deploy: Vercel

## Configuração

### Variáveis de Ambiente (Vercel)

- `DATABASE_URL`: Connection string do Supabase
- `SECRET_KEY`: Chave secreta para sessões

### Inicializar Banco de Dados

Execute o script SQL no Supabase SQL Editor ou use `python database.py` localmente.

## Deploy

O projeto está configurado para deploy automático no Vercel via GitHub.
